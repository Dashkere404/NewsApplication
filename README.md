# Новостная лента
Пользователь видит популярные за последние 30 дней новости, опубликованные New York Times. При нажатии на карточку новость открывается в браузере.
# Запуск
- Склонировать репозиторий на локальный компьютер в Android
- Запустить приложение на эмуляторе, либо на телефоне через провод
# Важное пояснение
На самом деле не рекомендуется прописывать личный API-ключ в коде, однако для быстрого запуска приложения было принято решение добавить личный API-ключ прямо в код.
